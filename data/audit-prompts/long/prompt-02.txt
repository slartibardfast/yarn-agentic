Compilers translate high-level source code into target machine code through a sequence of well-defined phases. The frontend tokenizes the source text into a stream of lexemes, then parses that stream against the languages grammar to construct an abstract syntax tree. Semantic analysis decorates the tree with type information, resolves symbol references, and verifies that the program is well-formed according to the languages static rules. Modern compilers then lower the tree to an intermediate representation suitable for optimization, where transformations such as constant folding, dead code elimination, loop-invariant code motion, and inlining are applied to improve performance without changing observable behavior. Finally the backend selects target instructions, allocates registers, generates assembly, and emits an object
