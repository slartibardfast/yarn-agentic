User flagged "Knob it" as not recommended in British English (carries unintended profane/slang connotations in UK usage).

**Why:** The user is British (David Connolly). Phrasings that read as neutral in US English can carry unintended slang or profane connotations in UK English.

**How to apply:** Avoid phrasings like "knob it", noun-as-verb shortenings that read US-informal. Prefer fuller verbs: "gate it behind an env flag", "make it configurable", "add a flag for it". When summarising tradeoffs, write in plain neutral English; do not adopt US-tech-startup vernacular.
